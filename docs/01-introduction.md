# Introduction

This guide provides comprehensive documentation for developing, packaging, and deploying custom applications on mPower devices running firmware version 7.0 or later. Custom applications extend the functionality of your mPower devices by allowing you to run user-created software alongside the standard features.

You manage custom applications through the mPower application framework, which handles installing, updating, starting, stopping, and monitoring your applications. This framework integrates with MultiTech Device Manager for remote deployment and management, making it possible to manage applications across fleets of devices.

## Who This Guide Is For

This guide is intended for developers and technical personnel who need to create custom applications for mPower devices. Readers should have experience with embedded Linux development and familiarity with at least one supported programming language.

## What You'll Learn

-   Structuring and configuring custom applications
-   Understanding the application lifecycle and management framework
-   Packaging and deploying applications
-   Using advanced features for production deployments
-   Troubleshooting and debugging common issues
